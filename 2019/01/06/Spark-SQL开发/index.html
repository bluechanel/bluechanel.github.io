Spark-SQLејҖеҸ‘

RDDе’ҢDataFrame

е·Ұдҫ§зҡ„RDD[Person]иҷҪз„¶д»ҘPersonдёәзұ»еһӢеҸӮж•°пјҢдҪҶSparkвҖҷжЎҶжһ¶жң¬иә«дёҚдәҶи§ЈPersonзұ»зҡ„еҶ…йғЁз»“жһ„гҖӮиҖҢеҸідҫ§зҡ„DataFrameеҚҙжҸҗдҫӣдәҶиҜҰз»Ҷзҡ„з»“жһ„дҝЎжҒҜпјҢдҪҝеҫ—SparkSQLеҸҜд»Ҙжё…жҘҡзҡ„зҹҘйҒ“иҜҘж•°жҚ®йӣҶдёӯеҢ…еҗ«йӮЈдәӣеҲ—пјҢжҜҸеҲ—зҡ„еҗҚз§°е’Ңзұ»еһӢжҳҜд»Җд№ҲпјҢDataFrameеӨҡдәҶж•°жҚ®зҡ„з»“жһ„дҝЎжҒҜпјҢеҚіschemaгҖӮRDDжҳҜеҲҶеёғејҸзҡ„JavaеҜ№иұЎзҡ„йӣҶеҗҲгҖӮDataFrameжҳҜеҲҶеёғејҸзҡ„RowеҜ№иұЎзҡ„йӣҶеҗҲгҖӮDataFrameйҷӨдәҶжҸҗдҫӣжҜ”RDDжӣҙдё°еҜҢзҡ„з®—еӯҗеӨ–пјҢжӣҙйҮҚиҰҒзҡ„зү№зӮ№жҳҜжҸҗеҚҮжү§иЎҢж•ҲзҺҮпјҢеҮҸе°‘ж•°жҚ®иҜ»еҸ–д»ҘеҸҠжү§иЎҢи®ЎеҲ’зҡ„дјҳеҢ–гҖӮ

еёёз”ЁDataFrameз®—еӯҗ

Transformation еҗ«д№ү
filter(conditionExpr: String): DataFrame ж №жҚ®иҝҮж»ӨжқЎд»¶еҜ№ж•°жҚ®иҝӣиЎҢиҝҮж»ӨгҖӮеҰӮпјҡpeopleDf.filter(вҖңage > 15вҖқ)
groupBy(cols: Column*): GroupedData ж №жҚ®еҸӮж•°еҲ—иҝӣиЎҢеҲҶз»„гҖӮеҰӮпјҡdf.groupBy($вҖңdepartmentвҖқ).avg()
join(right: DataFrame, joinExprs: Column): DataFrame ж №жҚ®жҢҮе®ҡзҡ„иЎЁиҫҫејҸиҝӣиЎҢJoinгҖӮеҰӮпјҡ df1.join(df2, $вҖңdf1KeyвҖқ === $вҖңdf2KeyвҖқ)гҖӮ
sort(sortCol: String, sortCols: String*): DataFrame еҜ№жҢҮе®ҡзҡ„еҲ—жҺ’еәҸеҗҺпјҢиҝ”еӣһдёҖдёӘж–°зҡ„DataFrameгҖӮеҰӮпјҡdf.sort(вҖңsortcolвҖқ)гҖӮ
select(col: String, cols: String*): DataFrame жҹҘиҜўйғЁеҲҶеҲ—ж•°жҚ®пјҢз”ҹжҲҗж–°зҡ„DataFrameгҖӮеҰӮпјҡ df.select(вҖңcolAвҖқ, вҖңcolBвҖқ)гҖӮ
intersect(other: DataFrame): DataFrame еҜ№дёӨдёӘDataFrameеҸ–дәӨйӣҶгҖӮ
dropDuplicates(): DataFrame еҜ№DataFrameдёӯзҡ„ж•°жҚ®еҺ»йҮҚгҖӮ
Action еҗ«д№ү
collect(): Array[Row] иҝ”еӣһжүҖжңүж•°жҚ®зҡ„Rowж•°з»„гҖӮ
count(): Long еҜ№DataFrameдёӯзҡ„ж•°жҚ®и®Ўж•°гҖӮ
Head(int N):Row иҝ”еӣһеүҚNиЎҢж•°жҚ®гҖӮ
first(): Row иҝ”еӣһз¬¬дёҖжқЎRowж•°жҚ®гҖӮ
show(numRows: Int): Unit д»ҘиЎЁж јеҪўејҸжҳҫзӨәDataFrameдёӯзҡ„ж•°жҚ®гҖӮ
take(n: Int): Array[Row] иҝ”еӣһDataFrameдёӯзҡ„еүҚnжқЎж•°жҚ®гҖӮ
