Hive Shell命令

基本操作

  1. 创建数据库

    create database zhang;

  2. 进入数据库

    use zhang;

  3. 创建表

    • 普通表

    分隔符为，

    可使用show create table student;查看见表语句

  4. 添加数据的三种方式

    • 加载的方式

      load data local inpath '/bigdata/student.csv' overwrite into table student;

    • 通过hdfs的put方式

      hdfs dfs -put student.csv /user/hive/warehouse/zhang.db/student

    • insert的方式(低版本无法使用)

  5. 删除表

    drop table student;

  6. 删除库

    drop database if exists zhang cascade;

  7. 删除字段

    直接使用drop命令删除列时出现错误

    FAILED: ParseException line 1:37 cannot recognize input near ')' '<EOF>' '<EOF>' in column specification

    使用replace删除除id之外的列

    alter table student replace columns(id int);

  8. 修改字段

    修改字段test为int类型的test1

    alter table student change test test1 int

  9. 展示表的详细信息

    desc formatted student;

  10. 展示所有数据库

    show databases;

Hive的优缺点

  • 优点
    • 高可靠，高容错，HiveServer采用集群模式，支持超时重试
    • 语法类SQL，支持大量函数
    • 可扩展：可自定义函数和存储格式
    • 多接口：ODBC,PYTHON,BEELINE等
  • 缺点
    • 延迟高，依赖mapreduce
    • 不支持物化视图
    • 不支持存储过程
